# 🧮 Calculator Web Application | CodSoft Web Development Internship

This repository contains a responsive calculator web application developed as part of the **CodSoft Web Development Internship**. The project demonstrates the implementation of a clean user interface and interactive calculator functionality using HTML, CSS, and JavaScript.

The application performs basic arithmetic operations while providing a simple and user-friendly experience.

---

# 📌 Project Overview

The objective of this project was to build a fully functional calculator capable of performing common mathematical operations using client-side JavaScript.

The project helped strengthen core web development concepts including DOM manipulation, event handling, JavaScript logic, and responsive interface design.

---

# ✨ Features

- Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Decimal number support
- Clear (Reset) functionality
- Division-by-zero error handling
- Responsive calculator interface
- Interactive button-based input
